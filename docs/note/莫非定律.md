# 莫非定律

## 镜我效应

>一个人的自我观念是与他人交往中形成的,
>对自己的认识是他人对自己的看法的反映
>自我感觉来自于别人的思想,以及他人的态度

自己知道自己长什么样,需要照镜子,自己觉得自己帅都是别人夸的

## 自我服务偏见

>轻易的为自己的失败开脱,怨天尤人
>欣然理得的接受荣誉
>夸大对自己有利的信息,忽略不利的信息

## 沉锚效应(先入为主)

>人们做决策前都会被第一信息左右,第一信息就会像沉入海底的锚一样把你的思想固定在某处

根源在于接受的信息量太少,拓宽视野,不断学习,实践

解决办法

1.彻底无视之前信息

2.大量收集数据,全面分析
